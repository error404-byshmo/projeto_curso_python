# Plataforma de Curso Python PET

Uma plataforma educacional interativa desenvolvida com **Streamlit** e **Jupyter Notebooks**. O projeto foi desenhado para proporcionar um aprendizado ativo, onde o aluno lê a teoria e pratica o código no mesmo ambiente.

## Funcionalidades
- **Navegação Modular:** 8 módulos progressivos, do básico ao Pandas.
- **Ambiente Integrado:** Notebooks Jupyter renderizados diretamente na interface web.
- **Progresso Controlado:** Travamento de módulos para garantir a trilha de aprendizado.
- **Material de Apoio:** Link direto para download dos arquivos originais no Google Drive.
- **Interface Responsiva:** Visual moderno e limpo focado na experiência do usuário.

## Tecnologias Utilizadas
- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/) (Interface Web)
- [Nbconvert](https://nbconvert.readthedocs.io/) (Conversão de Notebooks)
- [Pandas](https://pandas.pydata.org/) (Manipulação de Dados)
